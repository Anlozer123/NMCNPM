# Hospital Management System (HMS)

Đây là dự án Hệ thống Quản lý Bệnh viện, giúp quản lý bệnh nhân, bác sĩ và lịch khám bệnh trực tuyến.

## Công nghệ:
* **Frontend:** ReactJS
* **Backend:** Node.js (Express)
* **Database:** SQL Server

## Cách chạy

### Cài đặt Backend
Mở terminal tại thư mục `server`:
```bash
cd server
npm install
npm run dev
```
### Cài đặt Frontend
Mở terminal tại thư mục `client`:
```bash
cd client
npm install
npm start
```
### Chức năng chính:
* Đăng ký, đăng nhập
* Xem lịch khám đối với vai trò bác sĩ
* Sẽ còn cập nhật thêm ...
